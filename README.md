# Yuo-Selfbot
Account nuker, discord selfbot for discord. (will revamp/fix soon.)

![](https://replit.com/@ArskyIsOnTop/Yuo-Selfbot)

# Steps
  1. Make a .env file
  2. Put this is the file (Look below)
      
     token=
     password= 
     prefix=    
.
  3. Then put in your info then start the selfbot by clicking Run
